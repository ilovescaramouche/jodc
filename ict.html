CSS, or Cascading Style Sheets, is a language used to style and format HTML documents. It allows web developers to control the appearance of web pages, including layout, colors, fonts, and spacing. CSS separates content from design, making websites easier to maintain and more visually appealing. This lesson introduces the purpose of CSS, its syntax, and its role in the presentation layer of a webpage.

CSS is composed of selectors and declarations. A selector targets HTML elements, while declarations define the styles to apply. Each declaration includes a property and a value, enclosed in curly braces. This lesson explains how to write CSS rules, the importance of declaration blocks, and how browsers interpret these rules to render styled content. It also covers the concept of style sheets and how they affect webpage layout.

There are three main types of CSS: inline, internal (embedded), and external. Inline styles are written directly within HTML tags, internal styles are placed in the `` section of an HTML document, and external styles are stored in separate `.css` files. This lesson compares their advantages and disadvantages, explains how to link external stylesheets, and demonstrates how each type affects webpage design and maintainability.

CSS rules consist of selectors and declarations. Classes are reusable selectors defined with a dot (`.`) and applied to HTML elements using the `class` attribute. This lesson explores how to create and apply classes, the difference between class and element selectors, and how to combine multiple classes. It also introduces the concept of specificity and how CSS rules are prioritized when multiple styles apply.

Selectors are used to target HTML elements for styling. This lesson covers various types of selectors: element/tag selectors, class selectors, ID selectors, universal selectors (`*`), group selectors (e.g., `h1, p`), and attribute selectors (e.g., `input[type="text"]`). Each selector type is explained with examples, and students learn when and why to use each based on the structure and needs of their webpage.

This lesson focuses on controlling the size and visibility of HTML elements using CSS properties. Students learn about `width`, `height`, `max-width`, `min-height`, `line-height`, and `visibility`. Examples show how these properties affect layout and user experience. The lesson also introduces the `display` property and its values (`block`, `inline`, `none`, etc.), and how visibility differs from display in terms of rendering and space allocation.
